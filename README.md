## 先搞清楚一件事：住宅 IP 和数据中心 IP 到底差在哪

**住宅 IP**（Residential IP）的定义很简单：这个 IP 地址由真实的互联网服务提供商（ISP，比如 AT&T、Comcast）分配给真实的家庭用户。它的 WHOIS 记录显示的是居民宽带信息，归属为某个具体的住宅地址。

**数据中心 IP** 则相反，归属是 AWS、Vultr、Linode 这类云服务商的机房，哪怕你在上面跑个代理，对外展示的流量特征仍然是"服务器环境"。

TikTok 的风控系统不仅看 IP 归属，还会综合设备指纹、行为轨迹、网络延迟特征来判断一个账号是否在"正常的家庭网络环境"下操作。一旦系统识别出你走的是机房出口，该触发的验证就会触发，该限流的就会限流。

## 怎么判断一个"住宅 IP"是真是假

这是个值得单独说的问题，因为市场上打着住宅 IP 旗号卖机房 IP 的商家不少。

几个实用的检测方法：

**ARIN WHOIS 查询**：访问 search.arin.net，输入 IP 地址查询注册信息。真正的住宅 IP 显示的 Organization 应该是运营商名称加"Internet Services"或"Residential"相关字样，而不是数据中心公司名。

**bgp.he.net 查询**：输入 IP 看 ASN 归属。AT&T 的 ASN 是 AS7018，Comcast 是 AS7922。如果显示的是 AWS（AS16509）、DigitalOcean（AS14061）这类，那就是机房 IP 无疑。

**在线黑名单检测**：用 MXToolbox 的 Blacklist Check 功能查询 IP 是否在垃圾邮件黑名单中。被大量营销账号用过的 IP，很可能同时被邮件系统标记过，TikTok 的风控里同样有这类信号。

**直接测试**：买之前要求商家先演示注册 Gmail——全程无需短信验证，才算初步过关。能正常登录 TikTok 且不触发异常检测，才算合格。
